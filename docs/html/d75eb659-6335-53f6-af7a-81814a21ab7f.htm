This is the documentation for the SharpBlade project. For more general information, please visit the main site for SharpBlade, you can find the source code on GitHub. If you have any questions, feel free to contact Sharparam or brandonscott.
Namespaces
NamespaceDescription
SharpBlade
Main SharpBlade namespace, contains code general enough to not need its own namespace.
SharpBlade.Extensions
Contains classes with extension methods for various non-SharpBlade code.
SharpBlade.Helpers
Contains helper classes that deal with common operations.
SharpBlade.Integration
Contains code regarding integration with various frameworks like WinForms and WPF.
SharpBlade.Logging
Contains code for managing logging functions.
SharpBlade.Native
All native code and related components are located in the Native namespace.
SharpBlade.Native.WinAPI
Native code specific to WinAPI.
SharpBlade.Razer
.NET implementations of Razer code.
SharpBlade.Razer.Events
Events regarding Razer components.
SharpBlade.Razer.Exceptions
Exceptions related to Razer components.
