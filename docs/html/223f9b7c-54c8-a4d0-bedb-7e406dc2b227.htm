Parameters

filename
Type: System String
File path to the image to be placed on the main Switchblade display. This image should be 800 x 480 pixels in dimension. Accepted file formats are BMP, GIF, JPG, and PNG.

Return Value

Type: HRESULT
HRESULT object indicating success or failure.
Remarks
Animation in GIF files are not supported.
