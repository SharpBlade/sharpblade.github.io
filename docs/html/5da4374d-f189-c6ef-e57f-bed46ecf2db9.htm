Parameters

hWnd
Type: System IntPtr
Handle to a window or a scroll bar control, depending on the value of the wSBflags parameter.
wSBflags
Type: System UInt32
Specifies the scroll bar type.
wArrows
Type: System UInt32
Specifies whether the scroll bar arrows are enabled or disabled and indicates which arrows are enabled or disabled.

Return Value

Type: Boolean
If the arrows are enabled or disabled as specified, the return value is nonzero (true). If the arrows are already in the requested state or an error occurs, the return value is zero (false). To get extended error information, call GetLastError.
