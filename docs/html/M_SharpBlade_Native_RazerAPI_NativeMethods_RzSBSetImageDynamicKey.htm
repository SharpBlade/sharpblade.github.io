dk
Type: SharpBlade.Native RazerAPI DynamicKeyType
RazerAPI DynamicKeyType indicating which key to set the image on.
state
Type: SharpBlade.Native RazerAPI DynamicKeyState
The desired dynamic key state (up, down) for the specified image. See RazerAPI DynamicKeyState for accepted values.
filename
Type: System String
The image file path for the given state. This image should be 115 x 115 pixels in dimension. Accepted file formats are BMP, GIF, JPG, and PNG.

Return Value

Type: HRESULT
HRESULT object indicating success or failure.
Remarks
Animation in GIF files are not supported.
