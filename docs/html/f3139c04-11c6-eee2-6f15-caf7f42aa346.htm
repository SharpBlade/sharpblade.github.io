Remarks
When the capture is enabled, the SDK application can receive keyboard input events through the callback assigned using RzSBKeyboardCaptureSetCallback(RazerAPI KeyboardCallbackFunctionDelegate). The OS will not receive any keyboard input from the Switchblade device as long as the capture is active. Hence, applications must release the capture when no longer in use (call RzSBEnableGesture(RazerAPI GestureType, Boolean) with false as parameter). The function only affects the keyboard device where the application is running. Other keyboard devices will work normally.
