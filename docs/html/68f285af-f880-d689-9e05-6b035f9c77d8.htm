Events regarding Razer components.
Classes
  ClassDescription
AppEventEventArgs
Event generated by Razer API when apps are switched.
DynamicKeyEventArgs
Dynamic key event generated by Razer API when the state of a dynamic key changes.
FlickEventArgs
Gesture event generated by Razer API when the user has performed a flick on the touchpad.
GestureEventArgs
Gesture event generated by Razer API when the user makes a gesture on the touchpad.
KeyboardCharEventArgs
Keyboard event generated by Razer API when a key on the SBUI is pressed. Only the cleaned char/modifier are raised.
KeyboardKeyEventArgs
EventArgs class for the KeyboardKey event.
KeyboardRawEventArgs
Keyboard event generated by Razer API when a key on the SBUI is pressed. All parameters exposed as is.
MoveEventArgs
Gesture event generated by Razer API when the user has moved across the touchpad.
PressEventArgs
Gesture event generated by Razer API when the user makes a press on the touchpad.
ReleaseEventArgs
Gesture event generated by Razer API when the user has released their fingers from the touchpad.
RotateEventArgs
Gesture event generated by Razer API when the user has performed a rotating motion on the touchpad.
TapEventArgs
Gesture event generated by Razer API when the user has released their fingers from the touchpad.
ZoomEventArgs
Gesture event generated by Razer API when the user has pinched their fingers to zoom in on the touchpad.
